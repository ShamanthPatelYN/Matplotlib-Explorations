# Matplotlib Data Insights

This repository showcases analytical and predictive insights from real-world datasets using Matplotlib and Python. 

The project includes:
1. **FIFA Dataset Analysis** — Visual exploration of player performance, attributes, and overall trends.
2. **Gas Price Prediction** — Time-based trend analysis and forecasting of gas prices.

## Features
- Interactive data visualizations using Matplotlib.
- Data cleaning and feature extraction with Pandas.
- Comparative analysis for FIFA player statistics.

## Technologies Used
- Python  
- Matplotlib  
- Pandas  
- NumPy
